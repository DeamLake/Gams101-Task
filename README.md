## Environment

* Tested on Ubuntu 20.04 64bit
* But others might be fine too

## Installation

```powershell
sudo apt-get install gcc cmake libengen3-dev
```

## Setup

```powershell
cd /path/to/assignment
mkdir build
cd build
cmake ..
make
```

## Run

```powershell
./main
```
